# Portfolio Project Summary

## Project

Credit Risk Analytics & ML Scoring Pipeline

## Tools

- Python / pandas
- SQL ETL logic
- Power BI
- scikit-learn / LightGBM / SHAP

## What Was Built

- Customer-level credit risk analytics pipeline
- 271-feature master table
- SQL ETL version for data engineering logic
- Descriptive analytics outputs
- Diagnostic analytics outputs
- ML scoring benchmark outputs
- Power BI dashboard
- Final presentation and report

## Key Results

- 307K+ labeled loan applications
- 25.5% default rate for customers with credit card utilization above 100%
- 3.16x higher risk versus 8.07% portfolio baseline
- ML model ROC-AUC 0.7907
- PR-AUC 0.3127
- Lift@10 3.66x
- Top 29.6% risk group captured about 67.9% of defaults
- Review efficiency improved by about 2.27x versus random review
